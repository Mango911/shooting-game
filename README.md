# 打飞机小游戏 🚀

一个用纯JavaScript制作的经典打飞机游戏，具有现代化的界面设计和流畅的游戏体验。

## 游戏预览

![游戏截图](./preview.png)

## 游戏特性

- 🎮 **流畅的操作体验** - 支持WASD键和方向键移动
- 🔫 **射击系统** - 空格键射击，带有射击冷却时间
- 👾 **动态敌机生成** - 敌机随机生成，难度逐渐增加
- ⭐ **背景星空效果** - 动态星空背景增强游戏氛围
- 💥 **爆炸粒子效果** - 击中敌机时的爆炸动画
- 🏆 **计分系统** - 实时分数显示和生命值管理
- ⏸️ **暂停功能** - 按P键暂停/继续游戏
- 📱 **响应式设计** - 适配不同屏幕尺寸

## 游戏操作

| 操作 | 按键 |
|------|------|
| 移动 | WASD 或 方向键 |
| 射击 | 空格键 |
| 暂停/继续 | P键 |

## 如何运行

1. 克隆或下载项目到本地
2. 用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮开始游戏

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画效果
- **JavaScript** - 游戏逻辑和交互
- **Canvas API** - 游戏渲染

## 游戏机制

- **玩家飞机**: 可以移动和射击，有3条生命
- **敌机**: 从屏幕上方随机生成，逐渐增加生成速度
- **子弹**: 玩家发射的子弹可以消灭敌机
- **碰撞检测**: 子弹击中敌机或玩家撞到敌机的碰撞处理
- **分数系统**: 击毁敌机获得分数，游戏结束显示最终分数
